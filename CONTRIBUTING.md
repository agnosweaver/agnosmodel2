## Why contributions are limited
The design of this project is **configuration-driven** and may later be supported by a **cloud platform** that hosts official as well as community driven wrappers, plugins, and integrations.  
To keep the core clean, stable, and predictable, changes to the main repository are tightly controlled.  
This is to ensure long-term maintainability and a solid foundation which others can build upon. If the foundation keeps changing frequently, the structures built on top of it will collapse!
